# React Template

## これは何？

プログラミング課題用のReactのプロジェクトテンプレートです。

下記のいずれかの課題を実施するために用意されています。

## Usage

課題を解く際には、本レポジトリをforkして開発を進めてください。

### セットアップ

```
$ npm install
$ npm run build
$ npm start
```
